# @vividcodeai/embedded-serial-mcp-linux-arm64

Prebuilt binary for Linux ARM64. This is a platform dependency of `@vividcodeai/embedded-serial-mcp`.

**Do not install this package directly.** Install the parent package instead:

```bash
npm install @vividcodeai/embedded-serial-mcp
```
